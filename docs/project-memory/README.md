# 染色宝项目记忆库

这个目录是染色宝 Product Studio 的项目专属记忆库，用来给后续线程、开发同事和 agent 快速接上上下文。

## 使用规则

- 先读本目录，再读 `desktop-app/docs/mac-win-collaboration-v1.md`。
- 这里只记录项目事实、版本冻结点、模型接入决策和测试结论。
- 不记录 API Key、账号、cookie、登录态、运行时日志、用户隐私数据。
- 每次完成重要功能、打包、测试、Git tag 或模型策略变更后，都同步更新这里。

## 当前记忆文件

- `current-state.md`：当前项目状态和 Windows 基线。
- `media-generation.md`：图片 / 视频模型接入决策。
- `local-byok-edition.md`：个人本地版 BYOK/API Key 策略。
- `ui-style-rules.md`：桌面端原始 UI 风格和新增模块规则。
- `agent-development-rules.md`：多 agent 分工开发规则。
- `handoff-rules.md`：后续协作和接手规则。
